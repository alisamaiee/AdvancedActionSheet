### Cocoapods

AdvancedActionSheet is available through [CocoaPods](http://cocoapods.org). Simply add the following to your Podfile:

```ruby
use_frameworks!

target '<Your Target Name>' do
  pod 'AdvancedActionSheet'
end
```
